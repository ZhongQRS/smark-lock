# smart-lock
基于stm32f407 && UCOS III 开发的智能锁

此项目用于学习

oled引脚接线
 ---------------------------------------------------------------
             GND       接地
             VCC       5V/3V
             D0        PC6
             D1        PC8
             RES       PC11
             DC        PE5
             CS        PA6           
----------------------------------------------------------------

蓝牙模块引脚接线
 ---------------------------------------------------------------
             GND       接地
             VCC       5V/3V
             RX        PA10
             TX        PA9
             其余引脚不接
----------------------------------------------------------------

薄膜矩阵键盘
----------------------------------------------------------------
            PB7 PA4 PG15 PC7 行
            PC9 PB6 PE6 PA8	 列
----------------------------------------------------------------

lock开关 
----------------------------------------------------------------
            PD6
----------------------------------------------------------------
